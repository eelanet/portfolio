About Me

Let me tell you a few things about me...

BIO

I am a Business Information Systems student at JAMK University of Applied Sciences. I have studied programming and web development for 2.5 years. My passion is in both frontend and backend development.

Web Development - Frontend

I've spent most time in learning modern frontend web development. I have skills with HTML5, CSS3 and Javascript. Angular is the framework that I'm most familiar with. I like designing and creating beautiful websites and apps. I've also used SASS, Bootstrap and Foundation for styling.

Web Development - Backend

My backend development knowledge is almost at the same level with frontend. I'm familiar with Nodejs and Express and I've build multiple RestAPIs. I've connected my backend usually to MongoDb database with Mongoose and MySQL database with Sequelize.

Mobile Development

I use web development technologies also when developing mobile apps. I have experience with PWA and native mobile apps. Last spring I was part of two 5 months long NativeScript + Angular school projects.
